Mi Dashboard

📘 UI Framework - Success

Framework ligero, accesible y moderno creado por Fernando Salguero para prototipado rápido, dashboards y sistemas de administración.

📦 Instalación

✅ CDN (recomendado)

<link rel="stylesheet" href="https://www.fernandosalguero.com/cdn/assets/css/framework.css">
<script src="https://www.fernandosalguero.com/cdn/assets/javascript/framework.js" defer></script>

🗂️ Carpeta local

/assets
├── css/
│   └── framework.css
├── javascript/
│   └── framework.js

🎛️ Variables globales

:root {
  --primary-color: #5b21b6;
  --sidebar-bg: #fff;
  --text-color: #333;
  --icon-color: #5b21b6;
  --border-radius: 12px;
  --font-family: 'Segoe UI', sans-serif;
}

Podés sobreescribirlas directamente desde un archivo `theme.css` o en un bloque luego del CSS del framework.

🧩 Clases disponibles

  • .layout – contenedor base general
  • .sidebar y .sidebar.collapsed – menú lateral
  • .navbar – encabezado superior
  • .card – tarjetas visuales
  • .form-modern – formularios con estilo moderno
  • .input-icon – input con íconos
  • .btn, .btn-aceptar, .btn-cancelar, .btn-info
  • .floating-alert – alertas flotantes
  • .modal – ventanas emergentes (modales)
  • .card-grid grid-[n] – sistema de columnas para tarjetas (2, 3 o 4)
  • .tabs, .accordion, .switch – componentes interactivos

📚 Componentes listos para usar

¡En la siguiente sección encontrarás cada componente con su explicación, personalización y ejemplo listo para copiar!

🎨 Botones

Clases: .btn, .btn-aceptar, .btn-cancelar, .btn-info

Botones semánticos listos para usar con colores distintos y hover. Se pueden personalizar desde :root.

<button class="btn btn-aceptar">Aceptar</button>
<button class="btn btn-cancelar">Cancelar</button>
<button class="btn btn-info">Más info</button>
📦 Tarjetas

Clases: .card / .card-grid grid-2/3/4

Contenedores visuales para destacar contenido. Agrupá múltiples en columnas usando la clase .card-grid.

<div class="card">
  <h3>Título</h3>
  <p>Contenido</p>
</div>
✏️ Inputs con ícono

Clases: .input-icon

Inputs estilizados con íconos de Material. Se integran en .form-modern.

<div class="input-group">
  <label>Nombre</label>
  <div class="input-icon">
    <span class="material-icons">person</span>
    <input type="text" placeholder="Nombre completo">
  </div>
</div>
✅ Inputs validados

Detecta errores en campos vacíos o inválidos y muestra una leyenda en rojo. Se oculta al corregir.

<div class="input-group">
  <label>Edad (mínimo 18)</label>
  <div class="input-icon">
    <span class="material-icons">calendar_today</span>
    <input type="number" id="edad" min="18" required>
  </div>
  <small class="error-message hidden">Debés ser mayor de edad.</small>
</div>
📌 Select personalizado con búsqueda

Simula un select moderno con íconos y búsqueda opcional.

<div class="custom-select">
  <input type="text" placeholder="Buscar país..." onkeyup="filtrarOpciones(this)">
  <ul class="select-options">
    <li><span class="material-icons">flag</span> Argentina</li>
    <li><span class="material-icons">flag</span> Uruguay</li>
    <li><span class="material-icons">flag</span> Chile</li>
  </ul>
</div>
🔔 Alertas flotantes

Llamado JS: showAlert('success', 'Mensaje')

<button onclick="showAlert('info', 'Este es un mensaje informativo')" class="btn btn-info">
  Mostrar alerta
</button>
💬 Modal

Modales simples y centrados. Se abren con openModal() y cierran con closeModal().

<div class="modal hidden" id="modalEjemplo">
  <div class="modal-content">
    <h3>Título</h3>
    <p>Contenido del modal</p>
    <button onclick="closeModal()">Cerrar</button>
  </div>
</div>
🌀 Spinner Global

Clases: .spinner-overlay + .spinner | Funciones: showSpinner(), hideSpinner()

<div id="globalSpinner" class="spinner-overlay hidden">
  <div class="spinner"></div>
</div>
🧭 Tooltips

Agregá data-tooltip a cualquier elemento para mostrar un mensaje contextual.

<button class="btn" data-tooltip="Guardar cambios">
  Guardar
</button>
📑 Tabs

Dividí contenido en secciones fácilmente. Activación automática incluida.

<div class="tabs">
  <div class="tab-buttons">
    <button class="tab-button active" data-tab="uno">Uno</button>
    <button class="tab-button" data-tab="dos">Dos</button>
  </div>
  <div class="tab-content active" id="uno">Contenido uno</div>
  <div class="tab-content" id="dos">Contenido dos</div>
</div>
📂 Acordeones

Ideal para mostrar contenido expandible. Basado en <details>.

<details>
  <summary>Haz clic aquí</summary>
  <p>Contenido oculto</p>
</details>
🧱 Distribución de tarjetas

Utilizá .card-grid grid-2 / grid-3 / grid-4 para organizar en columnas responsivas.

<div class="card-grid grid-3">
  <div class="card">Tarjeta 1</div>
  <div class="card">Tarjeta 2</div>
  <div class="card">Tarjeta 3</div>
</div>
📄 Plantilla base del proyecto

Esta plantilla ya incluye el header, sidebar, layout y un formulario funcional. Ideal para comenzar un nuevo proyecto con Success sin configuraciones extras.

<!DOCTYPE html>
                <html lang="es">
                
                <head>
                  <meta charset="UTF-8" />
                  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
                  <title>Mi Proyecto con Success</title>
                
                  <!-- Íconos de Material Design -->
                  <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet" />
                
                  <!-- Framework Success desde CDN -->
                  <link rel="stylesheet" href="https://www.fernandosalguero.com/cdn/assets/css/framework.css">
                  <script src="https://www.fernandosalguero.com/cdn/assets/javascript/framework.js" defer></script>
                </head>
                
                <body>
                
                  <!-- 🔲 CONTENEDOR PRINCIPAL -->
                  <div class="layout">
                
                    <!-- 🧭 SIDEBAR -->
                    <aside class="sidebar" id="sidebar">
                      <div class="sidebar-header">
                        <span class="material-icons logo-icon">dashboard</span>
                        <span class="logo-text">Success</span>
                      </div>
                
                      <nav class="sidebar-menu">
                        <ul>
                          <li onclick="location.href='index.html'">
                            <span class="material-icons">home</span><span class="link-text">Inicio</span>
                          </li>
                          <li onclick="location.href='documentacion.html'">
                            <span class="material-icons">description</span><span class="link-text">Documentación</span>
                          </li>
                        </ul>
                      </nav>
                
                      <div class="sidebar-footer">
                        <button class="btn-icon" onclick="toggleSidebar()">
                          <span class="material-icons" id="collapseIcon">chevron_left</span>
                        </button>
                      </div>
                    </aside>
                
                    <!-- 🧱 MAIN -->
                    <div class="main">
                
                      <!-- 🟪 NAVBAR -->
                      <header class="navbar">
                        <button class="btn-icon" onclick="toggleSidebar()">
                          <span class="material-icons">menu</span>
                        </button>
                        <div class="navbar-title">Mi Proyecto</div>
                      </header>
                
                      <!-- 📦 CONTENIDO -->
                      <section class="content">
                
                        <div class="card">
                          <h2>¡Hola! 👋</h2>
                          <p>Esta es una tarjeta de bienvenida para empezar tu diseño.</p>
                        </div>
                
                        <div class="card">
                          <form class="form-modern">
                            <div class="input-group">
                              <label>Correo</label>
                              <div class="input-icon">
                                <span class="material-icons">mail</span>
                                <input type="email" placeholder="ejemplo@correo.com">
                              </div>
                            </div>
                
                            <div class="form-buttons">
                              <button class="btn btn-aceptar" type="submit">Enviar</button>
                              <button class="btn btn-cancelar" type="button">Cancelar</button>
                            </div>
                          </form>
                        </div>
                
                      </section>
                
                    </div>
                  </div>
                
                </body>
                
                </html>
